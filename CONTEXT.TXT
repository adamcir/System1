  - build/kernel32-floppy.bin

Implemented modules in all branches:
- vga
- bootlog
- panic
- interrupts (skeleton)
- keyboard (polling implementation)

VGA status:
- Default color: LIGHT_GREY
- API:
  - vga_set_color(...)
  - vga_set_cursor(row, col)
  - vga_get_cursor(&row, &col)
  - vga_putc_at(row, col, c)
- Hardware text cursor is synchronized (port 0x3D4/0x3D5).
- Enter/newline cursor movement fixed.

Keyboard status:
- Polling via ports 0x64/0x60.
- Supports regular ASCII keys + Shift.
- Supports Backspace.
- Supports arrow Left and Right.
- Arrow Up and Down intentionally ignored.
- New API:
  - keyboard_take_key()
  - keyboard_take_char()

Kernel loop behavior:
- Polls keyboard continuously.
- Echoes typed chars to VGA.
- Backspace erases previous character on screen.
- Left/Right arrows move cursor.

Floppy stability fixes already in place:
- stage1/stage2 LBA->CHS register clobber fix.
- linker.floppy.i386.ld base aligned to 0x00010000.

Known warnings:
- Linker warns about RWX LOAD segments (accepted in current MVP).

Next technical priority:
- Replace interrupts skeleton with full IDT + PIC + IRQ1 keyboard handler.
