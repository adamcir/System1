Module model (current):

  - vga
  - bootlog
  - panic
  - keyboard
  - tty
  - signals
- Interrupts:
  - common logic: `kernel/common/modules/interrupts/interrupts_common.c`
  - arch-specific wrappers:
    - kernel/i386/modules/interrupts/interrupts.c
    - kernel/x86_64/modules/interrupts/interrupts.c
    - kernel/i386-floppy/modules/interrupts/interrupts.c
- Signals:
  - common wrapper/core:
    - `kernel/common/modules/signals/signals.c`
    - `kernel/common/modules/signals/signals_core.c`
  - arch headers:
    - kernel/i386/modules/signals/signals.h
    - kernel/x86_64/modules/signals/signals.h
    - kernel/i386-floppy/modules/signals/signals.h
  - API:
    - `signal_raise(int signal)`
    - `HW_RESET` (`0`)
    - `HW_PWR_DOWN` (`1`)

TTY status:
- Shared `tty` module is active for all 3 outputs.
- `kmain_*` uses `tty_run()` instead of duplicated input loop.
- TTY handles:
  - backspace
  - left/right
  - delete
  - insert toggle
  - tab (`\t`) as 4 spaces
  - printable chars
  - `CTRL+ALT+DEL` via `signal_raise(HW_RESET)` (implemented in `signals_core`)
  - `CTRL+ALT+BACKSPACE` currently has no signal action (validation trigger removed)

VGA status:
- Hardware cursor sync active (0x3D4/0x3D5)
- Scroll-up on overflow enabled
- Color list exposed in `vga.h` via `#define VGA_COLOR_*` (0..15), with enum aliases.
- Compatibility alias for yellow is present (`VGA_COLOR_YELLOW`, `YELLOW`).
- Text editing API:
  - vga_text_begin(...)
  - vga_text_putc(...)
  - vga_text_backspace()
  - vga_text_left()
  - vga_text_right()
  - vga_text_delete()
  - vga_text_toggle_insert()
- Behavior:
  - right arrow bounded by text model
  - Enter in middle splits text to next line
  - Insert toggles insert/overwrite mode
  - `Home`/`End` APIs exist in VGA module, but are not bound in TTY key handling

Keyboard status:
- IRQ1 path active, polling fallback optional
- Supported editor control keys:
  - Left, Right
  - Insert, Delete
  - Home, End (currently not bound to TTY actions)
  - Tab
  - Caps Lock (letters-only, `Shift XOR CapsLock`)
  - Numpad numeric: `0-9`, `.`, `+`, `-`, `*`, `/`, Enter
  - Numpad navigation mode (with NumLock/Shift inversion):
    - Left, Right, Insert, Delete
  - Numpad `7` and `1` stay numeric (no Home/End mapping)
  - Ctrl+Alt+Del
- Key constants include:
  - KEY_LEFT/KEY_RIGHT
  - KEY_INSERT/KEY_DELETE
  - KEY_CTRL_ALT_DEL

Bootlog note:
- `bootlog_core` includes `vga.h` directly; yellow color usage is now explicit and type-safe.
